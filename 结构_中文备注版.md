## Codex 使用规则

```text
查找图片素材       → res://images/
查找场景文件       → res://scenes/
查找脚本文件       → res://scripts/
查找数值配置       → res://resources/
查找项目说明       → res://docs/
查找目录规范       → res://结构.md
```

## 维护原则

```text
章节专属图片       → images/backgrounds/chapters/对应章节/
多个章节共用图片   → images/backgrounds/shared/
英雄动作图片       → images/characters/heroes/对应职业/sprites/
怪物动作图片       → images/characters/enemies/对应类型/
技能特效图片       → images/effects/skills/对应职业/
UI 图片            → images/ui/
场景文件           → scenes/
脚本文件           → scripts/
数值配置           → resources/
```

## 注意事项

```text
不要创建 assets/ 文件夹，图片统一放入 images/
不要把 .gd 脚本放进 scenes/
不要把 .tscn 场景放进 scripts/
不要把正式数值直接写死在 UI 脚本
不要手动修改 .godot/ 目录
不要使用“最终版2”“新建文件夹”等无意义名称
不要在路径和文件名中使用空格
所有文件夹和文件名统一使用小写英文和下划线
```
